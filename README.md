# Task Manager App

A responsive, client-side Task Manager application built with HTML, CSS, and Vanilla JavaScript.

## Features
* **Create:** Add new tasks with designated priority levels.
* **Read:** View all tasks in a list format, color-coded by priority.
* **Update:** Edit existing tasks and change their priority.
* **Delete:** Remove tasks from the list.
* **Data Persistence:** Tasks are saved to the browser's `localStorage`, ensuring data remains after a page refresh.
* **Responsive Design:** Fully functional and styled for both desktop and mobile devices.

## Setup Instructions

1.  **Clone the repository:**
    ```bash
    git clone <your-github-repo-url>
    ```
2.  **Navigate to the project directory:**
    ```bash
    cd task-manager-app
    ```
3.  **Run the application:**
    Simply open the `index.html` file in your preferred web browser. No server setup or build process is required.

## Technologies Used
* HTML5
* CSS3 (Flexbox, Media Queries)
* JavaScript (ES6+, DOM Manipulation, LocalStorage API)
